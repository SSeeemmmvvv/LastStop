### Краткое описание приложения

LastStop - это мобильное приложение на Flutter, предназначенное для помощи пассажирам общественного транспорта в своевременном выходе на нужной остановке. Приложение решает проблему частых случаев, когда пассажиры просыпают свою остановку во время поездки, особенно в часы пик или при усталости.

Основные возможности приложения:
- Отображение карты с текущим местоположением и маршрутом
- Установка будильника для конкретной остановки
- Звуковое оповещение при приближении к выбранной остановке
- Простой и интуитивный интерфейс для быстрого выбора остановки

Приложение создано для пассажиров общественного транспорта: студентов, работающих людей, туристов и всех, кто регулярно пользуется автобусами, троллейбусами, трамваями или метро. Целевая аудитория - активные пользователи городского транспорта в возрасте от 16 до 65 лет, которые ценят свое время и хотят избежать стресса от опозданий.

### Описание экранов в макете

Макет содержит следующие экраны:

1. **Home Screen (Главный экран)**:
   - Логотип приложения "LS"
   - Мини-карта с текущим местоположением
   - Основная карта маршрута
   - Кнопка "CHANGE STOP" для выбора остановки

2. **Alarm Screen (Экран будильника)**:
   - Список доступных остановок
   - Настройки времени оповещения
   - Переключатель включения/выключения будильника
   - Кнопка сохранения настроек

### Минимальные требования к макету выполнены:
- Не менее 2 экранов (главный и экран будильника)
- Наличие базовых UI-элементов:
  - Кнопки (CHANGE STOP, переключатели)
  - Текстовые поля (названия остановок)
  - Заголовки (LS логотип, заголовки экранов)
- Мобильный формат (вертикальная ориентация)
- Экраны имеют осмысленные названия (Home, Alarm)
